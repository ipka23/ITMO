Rose Model Report - Web viewable version

A web viewable version of the example Rose model (generated by Rational Rose Web Publisher) is available, but is not installed.

To install:

  1. You will need to install WinZip onto your machine as the default extraction utility . If you do not already have a copy of WinZip, go to http://www.winzip.com/ to download a copy.

  2. Download the Web viewable Rational Rose model

    • A dialog box asking you to either save or open the zip file will appear. 

    • Choose open

    • Ensure that the "Use Folder Names" option is checked.

    • Extract all files to the top level folder for this web example. 
      If the default instructions were followed, your top level folder structure should look like:

      ... wyliecollegeexample  (UNZIP INTO THIS FOLDER)
      courseregistrationproject 
      wcsoftwareprocessweb

     

    • Answer "Yes" when prompted to replace files.
