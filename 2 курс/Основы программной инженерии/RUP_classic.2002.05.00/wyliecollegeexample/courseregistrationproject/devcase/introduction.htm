Purpose / Scope 

The purpose of this Development Case (see RUP Artifact: Development Case) is to explain which parts of the underlying Wylie College Software Process (WCSP) Development Case is applicable to this project and which parts are not.

In particular, this Development Case specifies which artifacts to produce in the Course Registration System Project,  by describing any deviations from the underlying Development Case set up for Wylie College Projects.

This Development Case is suitable for small to medium sized WEB based projects.
