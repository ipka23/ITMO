Notes on the Artifacts 

The project will produce one Status Assessment on the termination of each phase.  The results of each iteration will be assessed in separate Iteration Assessments. 

The Project Plan is part of the Software Development Plan. 

The Project Measurement Plan used is the Wylie College Standard Measurement Plan.

The Configuration Management Plan is considered part of The Software Development Plan, but it is grouped under and described in The Configuration and Change Management Discipline.
