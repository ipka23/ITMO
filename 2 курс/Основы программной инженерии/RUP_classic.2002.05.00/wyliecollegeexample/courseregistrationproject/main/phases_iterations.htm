Phases and Iterations

The plan of the various phases and iterations in the project is part of the Project Plan, which in turn is enclosed in the Software Development Plan.  This page contains a high level view of the schedule for this project with regards to iterations with major milestones.

This project is planned to release two versions of the software.  When starting Transition phase for release #1, an overlapping third construction iteration is started simultaneously, followed by a second Transition iteration.  This is to ensure customer satisfaction, since we expect the customer needs to change along the project.

To read more about the project plan, follow this link to the Software Development Plan.
