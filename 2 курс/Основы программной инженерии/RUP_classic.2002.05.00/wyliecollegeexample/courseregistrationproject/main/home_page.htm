 CRS Project Web Site Launched (4/2/99)

The CRS project team launched this project web site on April 1st, 1999.  This site is chartered to be the single source of  information for development of the Course Registration System.  It provides the project team with the ability to share and communicate development progress, status, and ideas.  This web site may also be utilized for communicating project status to the Wylie College management and customer relations.
Course Registration Project leads the way with process improvement at Wylie College (5/12/99)

Since July 1st 1998, a team led by our Chief Process Engineer, Simon Jones, has been working with Rational Software to define and develop a new standard software development process for projects developed at Wylie College. The basis for the improvement is the organizational wide adoption of Rational Unified Process (RUP) as the underlying Software Process Framework. The result of this work is a web site "Wylie College Software Process Web (WCSP)", describing the preferred RUP configuration for Wylie College' projects. This project web site is instantiated from WCSP. The experiences gathered from the Course Registration System Project will be used to improve the underlying organizational process further. 
Test Automation will be evaluated on CRS project (05/19/99)

The management of Wylie College IT Department has decided to test Rational Software's Test Automation product Rational Robot. This is a Pilot Project for Test Automation at Wylie College, and the project is required to follow the organizational standards for manual test routines parallel to evaluating the new product.
