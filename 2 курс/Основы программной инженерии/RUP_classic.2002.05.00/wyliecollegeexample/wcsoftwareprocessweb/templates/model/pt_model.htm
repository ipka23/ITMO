Installing Templates 

The RUP Artifact: Analysis Model, RUP Artifact: Use Case Model and RUP Artifact: Design Model for your project should be created using Rational Rose using the Wylie College Analysis / Design Model Template.  This template can be installed into Rational Rose as follows:

  1. Launch Rational Rose.

  2. A popup window titled "Create New Model" should appear.

  3. Select "Make New Framework" and click "OK".

  4. Fill in the dialog as follows:

    • Framework Name: Wiley College RUP Framework

    • Model File: Browse to this Organization Web's top folder, and down into templates\ana_desi\model_framework.  

      • Select the file (Wylie College RUP Framework.mdl).

    • Start Diagram: Select "Logical View / Welcome"

    • Documentation File: Select the file (Wylie College RUP Framework.rtf).

    • Icon File: Select the file (Wylie College RUP Framework.ico).

    • Click on "Finish".

The new framework may now be used as the basis for any project models you now wish to create.
