Templates 

The RUP Business Use-Case Realization Specification documents for your project can be created using the Rational« RequisiteProÖ Business Use-Case Realization Specification Document Type.  This uses the RUP Template: Business Use-Case Realization Specification as its outline.  To ensure proper creation the document should be created from inside the project's Rational« RoseÖ Use-Case Model or the Rational« RequisiteProÖ project.  

Integrated Use-Case Management is used to keep both the Rational« RoseÖ Use-Case Model and the Rational« RequisiteProÖ project seamlessly synchronized with each other (see the RUP Tool Mentor: Managing Use Cases with Rational Rose« and Rational RequisitePro« for instructions on using Integrated Use Case Management).
