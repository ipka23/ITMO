Purpose 

[The purpose of this template Rational Unified Process Development Case (see RUP Artifact: Development Case) is to facilitate the production of project and organizational development cases (see RUP Concept: Process Configuration)  when adopting the Rational Unified Process. 

This template is provided as part of the larger Process Configuration Template which is intended to be used in conjunction with, and is complementary to, the Rational Unified Process (RUP) online content.  To this end the template has been developed to the same standards as the RUP and includes links to the online RUP material. 

It extends the standard development case template to:

  • provide a multi-page development case template with project web compatible tree-browser
  • split up the elements of the development case into separately manageable components
  • group the artifacts to make the development case less intimidating
  • interact with the other elements of the Process Configuration Template]
