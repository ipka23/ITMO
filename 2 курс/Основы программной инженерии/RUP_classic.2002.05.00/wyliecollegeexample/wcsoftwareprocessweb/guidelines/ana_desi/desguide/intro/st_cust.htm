Background 

This web site is supplied as-is as an example set of analysis and design guidelines.  The site is designed to be used as the basis for individual company / project guidelines published as part of a Rational Unified Process Project Web-Site.  To this end certain customizations have been built into the pages.

Guidelines' Banner 

The banner down the right hand side of the page is a bit map that can be replaced with your own company / department details.

Revised Standards 

The standards themselves can be easily edited to reflect local standards.  These standards are not a formal released product and will not be upgraded by Rational.

Some sections need to be completed for your organization (See Standard: Design Classes: Design Stereotypes or Standard: Attributes in Design: Value Types) as these standards are specific to the programming language and development environment.
