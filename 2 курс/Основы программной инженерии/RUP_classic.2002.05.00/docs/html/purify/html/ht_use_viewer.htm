Using the Purify Viewer

Purify, by default, opens the Viewer to display error data when you run an instrumented program file.

Notes:

  • For information on a Purify message, select the message in the Viewer, then click the Explain button.

  • Context-sensitive Help is available for Viewer components such as menu items and controls. Select Help > On context, then click on the element.

  • For information about applying options in the Viewer, click

  • To display a different set of error data from a view file representing another run, select File > Open and choose a file. You can also open the Viewer to display a specific view file using the -view option.
