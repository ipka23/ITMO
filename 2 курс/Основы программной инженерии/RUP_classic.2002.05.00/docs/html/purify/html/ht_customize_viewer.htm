Customizing the Purify Viewer

When you exit your first Purify session, Purify automatically creates a .purify.Xdefaults file in your home directory. You can edit this file directly from the Viewer (as well as in a text editor) in order to customize the Viewer. For example, you can:

  • Define the number of lines of source code displayed in a message

  • Change background and highlight colors, and the size and color of fonts

  • Color code messages

Purify simplifies the task of customizing the Viewer by letting you edit the .purify.Xdefaults file directly from the Viewer:

  1. Select Options > Edit X resources to open the .purify.Xdefaults file.

  2. Edit the .purify.Xdefaults file using your editor.

  3. The .purify.Xdefaults file uses a typical .Xdefaults file format. For more information about editing an .Xdefaults file, see your X Window System Users Guide.

  4. Restart the Viewer for the changes to take effect.
