Using Rational ClearQuest with Purify

Rational ClearQuest is Rational Software's change request management system. If ClearQuest is installed at your site, you can log on to ClearQuest without leaving the Purify user interface.

To file a change request:

  1. Select the message you want to report. The message and the log file will automatically be included in the change request.

    Note:     This level of automation is available only if you are configured to access ClearQuest through its native interface. If you access ClearQuest through the Netscape browser, selecting the message has no effect.
     

  2. Click the Change Request button or select File > File a Change Request in the Viewer.
     

  3. Complete your change request in the ClearQuest interface as usual. See your ClearQuest documentation for further information.
