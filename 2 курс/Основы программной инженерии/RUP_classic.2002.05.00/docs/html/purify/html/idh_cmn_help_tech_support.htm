Contacting Technical Support

If you have a technical problem and you can't find the solution in online Help:

  1. Note the sequence of events that led to the problem and any program messages displayed.

  2. Email or call Technical Support.

  • If you email, include license information that appears in the log file.

  • If you call, have your license information ready. If possible, have the product running on your computer and the log file open.
