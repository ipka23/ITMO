About customizing Purify

You can customize Purify output to provide exactly the information you need, and to deliver it in the most efficient way. You can:

  • Annotate output

  • Customize Purify messages

  • Control message batching

  • Customize the thread summary message

  • Report status at exit

  • Mail output to team members

You can also customize the Purify Viewer and certain Viewer operations:

  • Customize the Purify Viewer

  • Customize the program controls

  • Customize Purify scripts

You can also:

  • Enable just-in-time debugging

  • Run shell scripts at exit

  • Manage cached object files

  • Select a configuration management system

  • Use custom memory management systems
