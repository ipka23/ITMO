Reload all (File menu, PureCoverage Viewer)

Reloads the data for all the PureCoverage data files (.pcv) currently displayed in the Viewer. PureCoverage discards the data that is displayed and loads updated data.

You can choose (in the Some PureCoverage data changed dialog box) to have PureCoverage reload changed .pcv files automatically when you run tests with the Viewer open. The Reload all command is provided for the times when you choose not to reload data automatically.
