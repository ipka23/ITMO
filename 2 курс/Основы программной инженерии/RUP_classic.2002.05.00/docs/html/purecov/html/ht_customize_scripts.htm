Using and customizing PureCoverage utility scripts

Purify includes the following utility scripts in the <purecovhome> directory:

  • pure_invoke_clearquest or pure_invoke_ddts starts your change request management system when you click the Change Request button in the toolbar. See Configuring a change request management system for details.
     

  • pure_submit_clearquest_defect is called by pure_invoke_clearquest. The script makes automated entries in fields in Rational ClearQuest. You can customize the script to enter the appropriate values in any fields defined in your group's ClearQuest schema.
     

  • pure_print sends an ASCII version of your PureCoverage run to the printer when you select File > Print.
     

  • pure_remove_old_files cleans your cache directory.
     

  • purecov_what_options <executable filename> tells you what PureCoverage options were used when the program was built.
     

To customize a PureCoverage utility script

  1. Copy the script to a separate directory.

  2. Edit the script.

  3. Set your path to include the path of the modified script. Make sure the new path is placed before the original path.
