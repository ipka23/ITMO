Options for debugging messages

Option

Default

-debug

no

Instructs PureCoverage to print debug messages during instrumentation.

 

 

 

-verbose

no

Instructs PureCoverage to print debug messages in verbose format during instrumentation.

 

 

 

-thread-trace-contexts

no

Prints out context switch messages.

 

 

 

-thread-trace-locks

no

Prints out information about rtlib lock usage.

 

 

 

-thread-debug-assertions

no

Prints out thread assertion messages.
