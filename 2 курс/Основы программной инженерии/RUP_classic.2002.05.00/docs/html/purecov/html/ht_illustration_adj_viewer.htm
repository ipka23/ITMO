How the Viewer displays adjustments

By default, the Viewer is set to display line data as modified by adjustments. It also displays the total number of adjustments for your program, and each directory, file, and function in it.

If you make or remove adjustments in the Annotated Source window, the Viewer updates its adjusted coverage data immediately, but does not re-sort the data automatically.

If you have marked adjustments in your source code, select Adjustments > Extract adjustments from all source files after instrumenting and running your program to display the adjusted coverage data.

Notes:

  • You can display additional columns in the Viewer to display more information about adjustments, or to display actual coverage in addition to, or instead of, adjusted coverage. Select View > Select columns.

  • If you mark all lines in a file dead, PureCoverage treats the file as if it contained only functions with no lines in them. The file still appears in the Viewer statistics.
