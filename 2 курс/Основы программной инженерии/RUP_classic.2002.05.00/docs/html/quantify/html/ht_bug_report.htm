Filing bug reports with Rational Software

To file a bug report for Quantify, please use the BUG_REPORT file in the quantifyhome directory.

Be sure to provide us with as complete a description of the problem as you can, including:

  • The compiler you are using, including the compiler version

  • Your compilation and linker options

  • Any test case material available

  • The log file

  • Any debug traces

  • Product, platform, and version numbers

  • The debugging dump; if the product crashed and produced one, and a description of what you were doing at the time

  • Any Quantify data

Email the report to technical support. See Contacting Technical Support for the address.
