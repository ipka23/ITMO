The basic steps

Quantify collects a complete, accurate set of performance data for your program and its components, including threads, and helps you find exactly where your program spends most of its time. To profile a program's performance:

  1. Collect performance data for your C/C++ program.

    (Quantify can also collect performance data for Java applications; for information, click ).
     

  2. Analyze the performance data.
     

  3. Run the program again and compare runs to find performance changes.
